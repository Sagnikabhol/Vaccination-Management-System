# AI-Powered Vaccination Management System

## Overview
A smart vaccination management system developed using Java, JSP, Servlet, JDBC, MySQL, HTML, CSS and JavaScript.

The system allows users to:
- Register and Login
- Book Vaccination Slots
- View Booking Details
- Get AI-based Vaccine Recommendations

---

## Features

- User Authentication
- Vaccination Booking System
- View Booking Details
- AI Recommendation Module
- Responsive Dashboard UI
- MySQL Database Integration

---

## Technologies Used

- Java
- JSP
- Servlet
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat
